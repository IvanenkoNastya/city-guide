# city-guide
Database management system "City Guide"

This is a coursework on "Data structures and algorithms"

The task:

There is a database containing information about districts, microdistricts, streets, institutions, theaters, museums, libraries, hotels, restaurants, transport.
Show a directory of institutions (theaters, museums, libraries, etc.).
Show a list of selected institutions in a specific district of the city, select an institution, find its address.
Find a type of transport that can get you to the institution you chose.

Deadline: 09.12.2025
